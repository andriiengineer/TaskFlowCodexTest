## Overview

You are provided with **TaskFlow**, a Kanban-style project management web application. Your task is to build a test automation framework and implement tests.

## Running the Application

```bash
# Navigate to the app folder
cd app

# Start a local server (choose one)
npx serve -p 8080
# OR
python -m http.server 8080

# Open in browser
http://localhost:8080
```

---

## Application Features

| Feature | Description |
|---------|-------------|
| **Kanban Board** | 5 columns with drag-and-drop tasks |
| **Task CRUD** | Create, edit, delete tasks via modal |
| **Multi-Select** | Ctrl+Click, Shift+Click selection |
| **Context Menu** | Right-click menu with actions |
| **Undo/Redo** | Full history with 50-step limit |
| **Filtering** | Filter by assignee and priority |
| **Search** | Real-time task filtering |
| **Keyboard Shortcuts** | Ctrl+Z, Ctrl+K, N, E, Delete, etc. |
| **Toast Notifications** | Success/error messages |
| **Drag and Drop** | Move tasks between columns |
